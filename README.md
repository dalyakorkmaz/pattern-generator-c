# pattern-generator-c
Console-based symmetric pattern generator written in C
This project is a console-based pattern generator written in C.
It was developed as part of a COM1002 lab assignment.
## Description
The program reads an integer from the user and generates a symmetric pattern using:
-Numbers
-'#' characters 
-Spaces for alignment
The pattern grows and shrinks symmetrically.

## Features
-Dynamic pattern generation based on input
-Symmetric design logic
-Modular function usage
-Clean console output formatting
